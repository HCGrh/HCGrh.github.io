  • vim编辑器

    • 命令模式: 复制,剪切,删除,粘贴,撤销,重做

    • 输入模式: 修改文件内容

    • 末行模式: 保存 退出 另存 查找 替换 设置

  • 命令模式进入到输入模式:

    • a 当前字符后输入

    • A 当前行尾输入

    • i 当前字符前输入

    • I 当前行首输入

    • o 当前行的下一行输入

    • O 当前行上一行输入

    • s 删除当前字符后输入

    • S 删除当前行后输入

    • HOME键 行首

    • END 行尾

  • 命令模式的操作

    • yy 复制一行 50yy

    • p 粘贴到下一行 P 粘贴上一行

    • dd 剪切/删除 100dd

    • ^行首 $行尾

    • y^ 复制当前字符到行首(不包含当前字符)

    • d^ 删除当前字符到行首(不包含当前字符)

    • y$ 复制当前字符到行尾(包含当前字符)

    • d$ 删除当前字符到行尾(包含当前字符)

    • gg 首行 G 尾行 500G

    • ygg 复制当前行到首行(包含当前行)

    • dgg 删除/剪切当前行到首行(包含当前行)

    • yG 复制当前行到尾行(包含当前行)

    • u撤销一步操作

    • ctrl+r 重做(恢复)

  • 末行模式

    • :w 保存

    • :q 退出

    • :wq 保存退出

    • :q! 强制退出

    • :w! 强制保存

    • :wq! 强制保存退出

    • :x 保存退出

    • :w /tmp/new.txt 另存

    • :10,20w /tmp/new.txt 10,20行另存

    • :e /tmp/new.txt 编辑某个文件

    • :e! 重新打开当前文件

    • :X 加密

    • 输入密码:123

    • 确认密码: 123

    • :wq

    • 取消密码

    • :X

    • 输入密码: 回车

    • 确认密码: 回车

    • :wq

    • 查找:

    • /ab n 下一个 N 上一个

    • ?ab N 下一个 n 上一个

    • 替换:

    • :g/^\s*$/d 删除只含有空格和制表符的行

    • :s/old/new/

    • :%s/ab/XX/gc

    • s 替换 switch

    • % 所有行

    • g 全局 globa

    • c 交互式

    • :3,5s/ab/XX/g

    • :$s/ab/XX/g 最后一行替换

    • :2,$s/ab/XX/g 2到最后一行替换

    • :%s/ //g 删除所有的空格

    • :%s/^ *//g 删除每行开头的所有空格

    • :%s/ // 删除每行第一个空格

    • :%s/^/#/ 注释掉所有行

    • :5d 删除第五行

    • :3,5d 删除3-5行

    • :g/xyz/d 删除含有xyz的行

    • :g/^$/d 删除空行

    • :g/^\s*$/d 删除所有空白行

    • s/o1d/new/g

    • 这个命令的含义是在当前行中替换所有出现的 “o1d” 为 “new” 。同样，“g” 标志表示全局替换。

  • ss -antlt |grep 3306 >> mysql.txt 找出想要的进程 追加到文件中

  • 特殊模式

    • 可视化模式 v进入

    • 光标选择 y复制 d剪切/删除 p粘贴

    • 可视化块模式 ctrl+v

    • 光标选择 y复制 d剪切/删除 p粘贴

    • ctrl+v 选中列 shift+i 输入# ESC键 2次

  • 多文件操作

    • vim aa.txt cc.txt tt.txt

    • :next 下一个

    • :prev 上一个

    • :last 最后一个文件

    • :first 第一个文件

    • :file 查看总数以及当前第几个

    • :files 所有文件信息

  • 同屏打开多个文件

    • vimdiff aa.txt cc.txt tt.txt
    • ctrl+w+w 切换文件
    • 横向 显示
      • vim -O aa.txt cc.txt tt.txt
    • 竖向 显示
      • vim -o aa.txt cc.txt tt.txt
