  • 用户管理:

    • 管理员: uid=0 root

    • 系统账号: uid=1-999

    • 普通账号: uid=1000-60000

    • 管理组: gid=0

    • 系统组: gid=1-999

    • 普通组: gid=1000-60000

  • 添加账号:

    • [root@localhost ~]# useradd abc1

    • [root@localhost ~]# vim /etc/passwd

      ![img](https://api2.mubu.com/v3/document_image/099a3ce6-1166-4c1a-8f2f-4c538933467b-18679580.jpg

    • 第一段: 用户名

    • 第二段: 密码占位符

    • 第三段: uid

    • 第四段: gid

    • 第五段: 描述

    • 第六段: 家目录

    • 第七段: shell

    • 用法

    • [root@localhost ~]# useradd -u 2000 abc2

    • -u 指定uid

    • [root@localhost ~]# useradd -u 2001 -g 2000 abc3

    • -g 指定gid

    • [root@localhost ~]# useradd -u 2002 -g 2000 -c ‘hello world’ abc4

    • -c 描述

    • [root@localhost ~]# useradd -u 2003 -g 2000 -c ‘hello world’ -d /mnt/abc5 abc5

    • -d 家目录

    • [root@localhost ~]# useradd -u 2004 -g 2000 -c ‘hello world’ -d /mnt/abc6 -s /bin/bash abc6

    • -s shell

    • [root@localhost ~]# useradd -M -s /sbin/nologin test

    • -M 不创建家目录

    • /sbin/nologin 非交互式shell

    • [root@localhost ~]# su - test 切换账号

  • 添加组:

    • [root@localhost ~]# groupadd redhat

    • [root@localhost ~]# vim /etc/group

    • ![img](https://api2.mubu.com/v3/document_image/d5b50819-fbe3-4f04-b099-023b0bc7c28b-18679580.jpg

    • 第一列: 组名

    • 第二列: 组密码占位符

    • 第三列: gid

    • 第四列: 用户名

    • 参数

    • [root@localhost ~]# groupadd -g 3000 east

    • -g 指定gid

  • 密码设置:

    • [root@localhost ~]# passwd robin

    • 或

    • [root@localhost ~]# echo 123 | passwd –stdin robin

  • 查看 passwod文件

    • vim /etc/shadow

    • ![img](https://api2.mubu.com/v3/document_image/5c1fb4e2-1a2f-4b7e-915a-3a177393c065-18679580.jpg

    • 第一列: 用户名

    • 第二列: 密码 有密码状态 无密码状态 !!密码锁定状态 !账号锁定 *该用户永久性不能登陆系统

    • 第三列: 密码的最后一次修改时间 (unix元年: 1970年1月1日)

    • 第四列: 密码最小时间(密码修改间隔)

    • 第五列: 密码的最大时间(密码有效期)

    • 第六列: 密码过期前警告时间

    • 第七列: 密码过期后账号过期时间

    • 第八列: 账号有效期

    • 第九列: 保留

    • 相同uid账号

  • 修改用户信息:

    • [root@localhost ~]# usermod -u 2500 abc1

    • [root@localhost ~]# usermod -g 3000 abc1

    • [root@localhost ~]# usermod -c “east east” abc1

    • [root@localhost ~]# usermod -s /bin/tcsh abc1

    • 修改家目录:

    • 方法1:

    • [root@localhost ~]# usermod -m -d /mnt/abc1 abc1

    • 方法2:

    • [root@localhost ~]# usermod -d /home/abc1 abc1

    • [root@localhost ~]# mv /mnt/abc1 /home/

    • 锁定账号

    • [root@localhost ~]# usermod -L robin

    • 解锁账号

    • [root@localhost ~]# usermod -U robin

  • 修改组:

    • [root@localhost ~]# groupmod -g 6000 east 修改gid

    • [root@localhost ~]# groupmod -n neweast east 修改组名

  • 修改密码:

    • [root@localhost ~]# passwd -l robin 锁定密码

    • [root@localhost ~]# passwd -u robin 解锁密码

    • [root@localhost ~]# passwd -S robin 查看密码状态

  • 删除用户

    • [root@localhost ~]# userdel -r abc1

  • 删除组

    • 删除组: 如果组里有其它账号,该组不能删除

    • [root@localhost ~]# groupdel abc1

  • 手动管理账号:

    • 1.groupadd -g 10000 west

    • [root@localhost mnt]# vim /etc/group

    • west: x :10000:

    • 2.useradd -u 10000 -g west jerry

    • [root@localhost mnt]# vim /etc/passwd

    • jerry: x :10000:10000::/home/jerry:/bin/bash

    • [root@localhost mnt]# mkdir /home/jerry

    • [root@localhost mnt]# cp /etc/skel/.bash* /home/jerry/

    • 3.passwd jerry

    • [root@localhost mnt]# echo 123 |passwd –stdin jerry

  • 附加组:

    • 主组一定是附加组 主组是用来定义创建文件时文件的所属组 附加组用来比对权限

    • [root@localhost ~]# usermod -a -G east je rry 在现有附加组基础上添加新的附加组

    • [root@localhost ~]# usermod -G new jerry 覆盖现有所有的附加组(除了主组对应附加组)

    • 添加组密码:

    • [root@localhost ~]# gpasswd boss

  • 用户配置文件:

    • /etc/login.defs uid范围 gid方位 密码的默认配置

    • /etc/default/useradd

    • HOME=/home 默认家目录

    • SHELL=/bin/bash 默认shell

    • SKEL=/etc/skel 默认环境变量模版地址

  • 查看当前用户

    • whoami #当前用户是谁

  • who i am #原始登录用户是谁

    • who 用户 终端 登录时间(来源主机)

  • w 当前用户 终端 来源主机 登录时间 cpu消耗 执行命令

    • last 登录用户的总信息

  • lastlog 所有账号最后一次登录时间记录
