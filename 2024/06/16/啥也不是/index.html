  • 虚拟终端（pts）快捷键

    • shift+ctrl+ + 扩大

    • ctrl + - 缩小

    • shift+ctrl + t 新标签

    • alt+ 1 2 3 切换标签

    • shift+ctrl+w 关闭标签

    • alt+ F4 关闭终端

  • tty切换

    • ctrl + alt F2 - F6 或者 ctrl + alt + Fn F2 - F6

    • 切换到桌面 alt + F1 或者 Fn + alt + F1

  • xshell 工具

    • 配置网卡文件

      • vim /etc/sysconfig/network-scripts/ifcfg-ens33

    • 重启网卡

      • systemctl restart network

    • 查询网卡信息

      • ip addr show ens33

      • ip a

      • ifconfig

  • 命令的组成

    • [root@localhost ~]# 命令 -[参数…..] [file…..]

    • root 当前用户

    • localhost 主机名

    • ~ 表示当前位置

    • #表示root用户 $ 表示 普通用户

  • 文件类型说明

    • - # rw——-# . #1 #root #root #1647 #7月 17 18:52# anaconda-ks.cfg

    • 第一列：文件类型

    • 第二列：基本权限

    • 第三列：是否在开启selinux环境下创建

    • 第四列：硬链接数

    • 第五列：文件拥有者

    • 第六列：文件所属组

    • 第七列：文件大小

    • 第八列：最后一次修改时间

    • 第九列：文件名

  • 文件类型

    • - 普通文件 regular

    • d 目录文件 directory

    • l 符号链接 symbolic 又称 软连接

    • s 套接字文件 socket unxi套接字和tcp套接字

    • p 管道文件 pipe

    • c 字符设备文件 character

    • b 块设备文件 block (硬盘 U盘 光盘)

  • ls 的参数

    • -l log 长格式信息

    • -a 显示所有文件

    • -h 按最大单位显示

    • -S 按大小排序(默认降序)

    • -r 逆序 reverse

    • -t 时间

    • -R 递归 recursive

    • -d 查看目录信息

    • -i 显示文件的inode号

    • -A 显示所有文件 不包括 隐藏文件

  • cd的使用说明

    • cd 切换路径 切换目录

      • 绝对路径：从/ 开始 的路径 是绝对路径

      • 相对路径：从当前目录开始 是相对路径

    • cd .. 返回上一层目录

    • cd . 刷新

    • cd ../.. 返回上层的上层

    • cd ~ 返回家目录 cd 回车 也可以

    • cd - 返回上次工作目录

    • pwd 显示当前目录

  • 获取帮助的方法‘

    • ls –help 获取 ls 的 参数

    • info ls

    • type ls

  • 命令分类:

    • 内部命令：cd是 shell的内嵌

    • 外部命令：cat是 /usr/bin/cat

    • 别名：ls 是 ’ls –color=auto‘ 的别名

      • 别名>外部>内部

      • 别名 > 外部

      • 别名> 内部

  • 查询和定义别名

    • alias 查询系统中所有的别名

    • alias la = ’ls -lah‘ 临时定义别名

    • unallas la 释放别名

  • 帮助文档位置

    • /usr/share/doc/ 的目录下

  • man手册

    • man 1 用户命令

    • man 2 系统调用

    • man 3 库调用

    • man 4 特殊文件

    • man 5 配置文件

    • man 6 游戏

    • man 7 杂项 man 7 man

    • man 8 系统命令

    • man -f passwd 查看解释

    • man -k passwd 模糊查询

  • / 下 的目录功能和作用

    • /bin 用户可执行目录

    • /sbin 系统可执行目录

    • /lib 32为库文件目录

    • /lib64 64位库文件目录

    • /proc 伪文件系统，内存中的信息和内核参数，硬件信息

    • /sys 伪文件系统，内存中的信息和内核参数，硬件信息

    • /usr 应用程序目录

    • /etc 配置文件目录

    • /var 日志和数据文件目录

    • /boot 启动程序目录以及启动时所需要的文件

    • /root root 的家目录

    • /home 普通用户的家目录

    • /dev 块设备和字符设备文件目录

    • /run 进程号与锁文件目录

    • tmp 临时文件目录

    • /mnt 其他挂载点

    • /opt 第三方软件目录

    • /media 媒介目录

    • /srv 服务器数据目录

  • 查看文件的命令

    • less 命令

      • less /root/anaconda-ks.cfg

    • more命令

      • more /root/anaconda-ks.cfg

      • 回车 下一行

      • 空格 下一页

      • b 上一页

      • q 退出

    • head命令 默认前十行

      • head /root/anaconda-ks.cfg

      • 查看 前二十行

        • head -n 20 /root/anaconda-ks.cfg

        • head -20 /root/anaconda-ks.cfg

    • tail命令 默认后十行

      • tail /root/anaconda-ks.cfg

      • 查看 后二十行

        • tail -n 20 /root/anaconda-ks.cfg

        • tail -20 /root/anaconda-ks.cfg

        • 使用 tail 查看 监控日志

        • tail -f /var/log/messages

    • cat 命令

      • 查看行号

        • cat -n /root/anaconda-ks.cfg -n 显示所有行号

        • cat -b /root/anaconda-ks.cfg -b 显示有效行号

        • 管道符 | 将前一个命令的输出 作为 后一个命令的输入

        • cat -n /root/anaconda-ks.cfg | les

    • 文件管理

      • touch 命令

      • 创建一个 文件

      • touch a.txt

      • 创建多个文件

      • touch a.txt b.txt

      • touch {a,b,c}{1,2,3}.txt

    • 同步时间

      • ntpdate ntp.aliyun.com

      • 修改时间 touch -d “2023-07-17 16:37:00” abc.txt

    • mkdir 创建

      • mkdir abc

      • mkdir abc bac

      • mkdir /usr/abc/abc/abc -pv

      • -p 递归创建

      • -v 显示过程

    • 剪切/拷贝

      • cp 源文件地址 目标地址

      • cp /root/abc.txt /tmp/

      • 拷贝过程中可以重命名

      • cp /root/abcd.txt /tmp/xxxx.txt

      • -r 递归拷贝

      • cp -r /root/abc/ /tmp/

      • cp -r /root/abc/ /tmp/yyyy

      • mv 源文件地址 目标地址

      • mv /root/abce.txt /tmp/

      • 重命名

      • mv /root/abc.txt /root/xyz.txt

      • 移动目录

      • mv /root/abcd /tmp/

    • 删除

      • rmdir 只能删除空目录

      • rmdir /root/a1

      • -f 强制删除

      • rm -f /root/abc2.tx

      • -r 递归删除

      • rm -rf /root/abc1

      • * 匹配所有字符

      • rm -rf /root/*.txt

    • 命令引用

      • 调用命令的结果作为文件名称使用

      • $(命令)

    • 查看时间

      • [root@localhost ~]# date | 2022年 07月 06日 星期三 11:34:49

      • [root@localhost ~]# date +%Y | 2022

      • [root@localhost ~]# date +%m | 07

      • [root@localhost ~]# date +%d | 06

      • [root@localhost ~]# date +%H | 11

      • [root@localhost ~]# date +%M | 36

      • [root@localhost ~]# date +%S | 26

      • [root@localhost ~]# date +%F | 2022-07-06

      • [root@localhost ~]# date +%T | 11:36:58

      • [root@localhost ~]# date +%F_%T | 2022-07-06_11:37:28

    • 查看Linux的发行版本

      • cat /etc/redhat-release

    • 查看cpu的型号

      • cat /proc/cpuinfo

    • 查看cpu个数

      • nproc
