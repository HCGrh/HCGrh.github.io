  • 虚拟终端（pts）快捷键

    • - shift+ctrl+ +  扩大

- ctrl + - 缩小

- shift+ctrl + t  新标签

- alt+ 1    2    3    切换标签

- shift+ctrl+w  关闭标签

- alt+ F4  关闭终端

  • xshell 工具

    • 配置网卡文件

      • vim  /etc/sysconfig/network-scripts/ifcfg-ens33

    • 查询网卡信息

      • - ip  addr   show   ens33

- ip a

- ifconfig

  • 文件类型说明

    • - \- # rw-------# . #1 #root #root #1647 #7月  17 18:52# anaconda-ks.cfg

- 第一列：文件类型

- 第二列：基本权限

- 第三列：是否在开启selinux环境下创建

- 第四列：硬链接数

- 第五列：文件拥有者

- 第六列：文件所属组

- 第七列：文件大小

- 第八列：最后一次修改时间

- 第九列：文件名

  • ls 的参数

    • - -l   log    长格式信息

- -a  显示所有文件

- -h  按最大单位显示

- -S  按大小排序(默认降序)

- -r  逆序 reverse

- -t  时间

- -R  递归   recursive

- -d  查看目录信息

- -i  显示文件的inode号

- -A  显示所有文件   不包括 隐藏文件

  • 获取帮助的方法‘

    • - ls  --help     获取 ls   的  参数

- info    ls

- type  ls

  • 查询和定义别名

    • - alias   查询系统中所有的别名

- alias  la = ’ls  -lah‘   临时定义别名

- unallas  la  释放别名

  • man手册

    • - man  1  用户命令

- man  2  系统调用

- man  3  库调用

- man  4  特殊文件

- man  5  配置文件

- man  6  游戏

- man  7  杂项   man  7  man

- man  8  系统命令

- man  -f  passwd   查看解释

- man  -k  passwd    模糊查询

  • 查看文件的命令

    • less 命令

      • less /root/anaconda-ks.cfg

    • head命令 默认前十行

      • - head /root/anaconda-ks.cfg

- 查看 前二十行

  -  head  -n 20 /root/anaconda-ks.cfg

  -  head  -20 /root/anaconda-ks.cfg

    • cat 命令

      • 查看行号

        • - cat -n  /root/anaconda-ks.cfg    -n  显示所有行号

- cat -b  /root/anaconda-ks.cfg   -b   显示有效行号

- 管道符    |       将前一个命令的输出 作为 后一个命令的输入

- cat -n /root/anaconda-ks.cfg | les

    • 同步时间

      • - ntpdate  [ntp.aliyun.com](http://ntp.aliyun.com/)

- 修改时间   touch -d  "2023-07-17  16:37:00"  abc.txt

    • 剪切/拷贝

      • - cp    源文件地址  目标地址

- cp /root/abc.txt /tmp/

- 拷贝过程中可以重命名

- cp /root/abcd.txt /tmp/xxxx.txt

- -r    递归拷贝

- cp -r  /root/abc/ /tmp/    

- cp -r  /root/abc/ /tmp/yyyy

- 

- mv    源文件地址   目标地址

- mv /root/abce.txt /tmp/

- 重命名

- mv /root/abc.txt /root/xyz.txt

- 移动目录

- mv /root/abcd /tmp/

    • 命令引用

      • - 调用命令的结果作为文件名称使用

- $(命令)

    • 查看Linux的发行版本

      • cat  /etc/redhat-release

    • 查看cpu个数

      • nproc
